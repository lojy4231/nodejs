
- 게시글 목록 조회
    제목, 작성자명, 작성 날짜를 조회
    작성 날짜 기준으로 내림차순 정렬
    - Request
        - Method: GET
        - URL: /post
    - Response
        - Status code: 200
        - Payload: { title: string, userId: string, date: number, post_unm: number.sort }[]

- 게시글 상세 조회
    제목, 작성자명, 작성 날짜, 작성 내용을 조회하기
    - Request
        - Method: GET
        - URL: /post/:postId
    - Response
        - Status code: 200
        - Payload: { title: string, userId: string, date: number, content: string }

- 게시글 작성
    제목, 작성자명, 비밀번호, 작성 내용 입력
    - Request
        - Method: POST
        - URL: /posting
        - Body: { title: string, userId: string, password: string,  content: string, post_unm: number }
    - Response
        - Status code : 201
        - Payload : { post_unm: number }

- 게시글 수정
    API를 호출할 때 입력된 비밀번호를 비교하여 동일할 때만 글이 수정되게 하기
    - Request
        - Method: PUT
        - URL: /post/:postId
        - Body: { title: string, userId: string, password: string,  content: string }
    - Response
        - Status code : 200
        - Payload : { message: string }

- 게시글 삭제
    API를 호출할 때 입력된 비밀번호를 비교하여 동일할 때만 글이 삭제되게 하기
    - Request
        - Method: DELETE
        - URL: /post/:postId
        - Body: { userId: string, password: string }
    - Response
        - Status code : 200
        - Payload : { message: string }
